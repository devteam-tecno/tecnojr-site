# TecnoJR Gradient System

The TecnoJR design system uses gradient-first styling for a modern, dynamic appearance.

## Gradient Philosophy

Gradients provide:
- **Visual Interest**: More engaging than flat colors
- **Brand Identity**: Purple-to-blue creates signature look
- **Depth**: Adds dimension to flat design
- **Motion**: Animated gradients guide user attention

## Available Gradients

### 1. Primary Gradient

Purple to blue diagonal gradient - most commonly used.

```css
.gradient-tecno-primary
```

**Usage:**
- Hero section backgrounds
- Primary CTA buttons
- Section highlights
- Card accents

**CSS:**
```css
background: linear-gradient(135deg, 
  oklch(0.6 0.26 295) 0%,    /* tecno-purple-500 */
  oklch(0.55 0.18 250) 100%  /* tecno-blue-500 */
);
```

### 2. Animated Gradient

Animated gradient for high-emphasis CTAs.

```css
.gradient-tecno-animated
```

**Usage:**
- Primary CTA buttons (hero, budget sections)
- Important call-to-action elements
- Hover states for major interactions

**CSS:**
```css
background: linear-gradient(90deg,
  oklch(0.6 0.26 295),
  oklch(0.55 0.18 250),
  oklch(0.6 0.26 295)
);
background-size: 200% 100%;
animation: gradient-shift 3s ease infinite;
```

**Animation:**
The gradient shifts horizontally on hover:
```css
@keyframes gradient-shift {
  0%, 100% { background-position: 0% 50%; }
  50% { background-position: 100% 50%; }
}
```

### 3. Diagonal Gradient

Perfect for icon wrappers and small accents.

```css
.gradient-tecno-diagonal
```

**Usage:**
- Icon backgrounds (feature cards, service cards)
- Small accent elements
- Badge backgrounds

### 4. Text Gradient

Gradient applied to text using background-clip.

```css
.text-gradient-tecno
```

**Usage:**
- Hero titles
- Section headings
- Important text highlights

**CSS:**
```css
background: linear-gradient(135deg,
  oklch(0.6 0.26 295),
  oklch(0.55 0.18 250)
);
-webkit-background-clip: text;
background-clip: text;
-webkit-text-fill-color: transparent;
```

## Gradient Utility Functions

Import from `@/lib/styles` for type-safe gradient usage:

```tsx
import { getGradientClass } from '@/lib/styles';

// Returns appropriate class name
getGradientClass('diagonal')   // 'gradient-tecno-diagonal'
getGradientClass('animated')   // 'gradient-tecno-animated'
getGradientClass('primary')    // 'gradient-tecno-primary'
getGradientClass('text')       // 'text-gradient-tecno'
```

## Usage Examples

### Button with Animated Gradient

```tsx
import { Button } from '@/components/ui/button';

<Button variant="gradient-primary" size="xl">
  Get Started
</Button>
```

### Icon with Diagonal Gradient

```tsx
import { Zap } from 'lucide-react';
import { getGradientClass } from '@/lib/styles';

<div className={cn(
  'h-12 w-12 rounded-lg flex items-center justify-center',
  getGradientClass('diagonal')

  <Zap className="h-6 w-6 text-white" />
</div>
```

### Text Gradient Heading

```tsx
<h1 className="text-gradient-tecno text-5xl font-bold">
  Welcome to TecnoJR
</h1>
```

### Card with Glow Effect

```tsx
import { getGlowEffectClass } from '@/lib/styles';

<div className="group relative">
  {/* Glow border on hover */}
  <div className={getGlowEffectClass({
    variant: 'primary',
    hoverOpacity: 50,
    blur: true,
  })} />
  
  {/* Card content */}
  <div className="relative bg-tecno-black-900 rounded-2xl p-6">
    Content here
  </div>
</div>
```

## Gradient Backgrounds

For section backgrounds and overlays:

```tsx
// Subtle background gradient
<section className="bg-linear-to-b from-tecno-black-950 to-tecno-black-900">
  {/* Content */}
</section>

// Radial gradient spotlight effect
<div className="bg-radial-gradient from-tecno-purple-500/20 via-transparent to-transparent">
  {/* Content */}
</div>

// Gradient overlay on images
<div className="relative">
  <Image src="..." alt="..." />
  <div className="absolute inset-0 bg-linear-to-t from-tecno-black-950/90 to-transparent" />
</div>
```

For drawing attention:

```tsx

  gradient-tecno-primary
  animate-pulse
">
  New feature!
</div>
```

## Best Practices

### DO ✓

- Use animated gradients sparingly (primary CTAs only)
- Maintain consistent gradient direction (typically 135deg)
- Layer gradients with opacity for depth
- Use gradients on important elements only

### DON'T ✗

- Don't use multiple animated gradients on one page
- Don't apply gradients to body text (readability)
- Don't combine too many different gradients
- Don't use gradients on small text

## Performance Considerations

Gradients are performant but keep in mind:

- **Animated gradients** use GPU acceleration (will-change: background-position)
- **Static gradients** have minimal performance impact
- **Multiple layers** may affect paint performance on low-end devices
- Use `background-attachment: fixed` sparingly (expensive)
